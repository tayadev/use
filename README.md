# use - a simple way to use modules in your shell scripts

## Example

```sh
#!/usr/bin/env use

use '@gh/tayadev/ShellStuff:textstyle.sh'

echo "${green}If this text is colored is works :)$reset"
```

## Installation
```sh
$ git clone https://github.com/tayadev/use
$ cd use
$ [sudo] make install
```
